# FUZO Landing

This is the standalone cinematic landing page for `fuzo.app`.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Deployment

Deploy this repository to Vercel and point the `fuzo.app` domain to it.
Ensure that the "Enter Experience" button in `src/App.tsx` points to `https://app.fuzo.app`.
