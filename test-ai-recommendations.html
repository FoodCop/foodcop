🎯 How It Works

1. User completes onboarding with dietary preferences and cuisine choices 2. AI system analyzes user profile preferences 3. Spoonacular API is queried with personalized search parameters 4. AI scoring algorithm calculates match scores based on: - Cuisine preference alignment (30 points) - Dietary restriction compliance (25 points) - Health score bonus (15 points) - Popularity factor (10 points) - Price consideration (10 points) - Preparation time (10 points) 5. Recommendations are sorted by match score 6. AI reasoning is generated for each recommendation 7. Results are displayed with interactive filtering options
