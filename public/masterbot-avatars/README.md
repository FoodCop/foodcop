## Instructions:

1. Save each of the 7 masterbot images to this directory with the exact filenames listed above
2. Ensure images are in PNG format for best quality
3. Run the SQL update script: `database/update_masterbot_avatars.sql`
4. Images will be served from `/masterbot-avatars/` URL path

## Image Requirements:

- Format: PNG (preferred) or JPG
- Dimensions: Square aspect ratio (e.g., 512x512, 400x400)
- Size: Under 1MB each
- Quality: High resolution for crisp display
